Title: Ubuntu 24.04 LTS release notes - Ubuntu release notes
More than a thousand packages have been updated to handle time using a 64-bit value rather than a 32-bit one, making it possible to handle times up to 292 billion years in the future. Ubuntu 24.04 LTS includes the new 6.8 Linux kernel that brings many new features. Detailed changes are reported in the Noble Kernel Release Notes post. GCC 🐄 is updated to the 14, binutils to 2.42, and glibc to 2.39. ... OpenJDK LTS 21 is the default in Ubuntu 24.04 LTS while maintaining support for versions 17, 11, and 8. OpenJDK 17 and 21 are also TCK certified, which means they adhere to Java standards and ensure interoperability with other Java platforms. In Ubuntu 24.04 LTS, the implementation changed so that settings from /etc/ssh/sshd_config (and snippets from /etc/ssh/sshd_config.d/) are read by a systemd generator to configure the ssh.socket unit accordingly. See the original discourse post for more details. The Pacemaker package was updated to version 2.1.6. This release is also provided for Ubuntu 22.04 LTS via the Ubuntu Cloud Archive. Open vSwitch has been updated to the 3.3.0 release. Open Virtual Network has been updated to the 24.03 release. These releases are also provided for Ubuntu 22.04 LTS via the Ubuntu Cloud Archive.

Title: Ubuntu release cycle | Ubuntu
Ubuntu releases a new version every six months. Releases of Ubuntu get a development codename ('Resolute Raccoon') and are versioned by the year and month of delivery – for example, Ubuntu 26.04 was released in April 2026. They provide cutting-edge features and hardware support every six months, but with only 9 months of updates. For long-term stability, production environments should use the LTS version, while interim releases suit those prioritizing speed and rapid feature testing. At the end of the 10 year ESM period, the Legacy add-on can be purchased to cover additional 5 years of the lifetime of Ubuntu LTS, giving a total of 15 years security maintenance and support for LTS releases. Title: Ubuntu 26.04 LTS: What's New Since Ubuntu 24.04?
While you could already Deb packages from the Ubuntu repositories (and outside of it) via App Center in 24.04 LTS, you couldn’t manage them. Well, in 26.04 LTS, you can! The Manage section has gained a filter to see installed Deb software individually or alongside snaps. This is part of a wider effort from Canonical to create a single, unified way for managing software on Ubuntu, irrespective of its source. App Center can also update some Deb apps alongside Snaps. You can install the Software & Updates tool on Ubuntu 26.04 from the repos, so it’s not gone-gone. If you upgrade from 24.04 LTS, you’ll still have it installed (Ubuntu rarely force removes software on upgrades). The Linux 7.0 kernel will be backported to 24.04 as the final HWE update in summer 2026. Another kernel change of note is the linux-lowlatency package, which is now retired. If you relied on it for audio or low-latency work the replacement is lowlatency-kernel, a userspace package for tuning at boot via GRUB using the generic kernel. On the graphics driver side, Ubuntu 26.04 LTS ships with Mesa 26.0.3 open-source drivers and sees the default NVIDIA’s proprietary driver at 590.x, though older versions remains available for those who need them – but the lack of user-facing GUI makes it tricker to switch. Naturally, plenty more apps, packages, libraries and foundational tooling have seen successive updates since 24.04 too, and those may include their own new features.
